Galatea

Cette page est une démonstration d'une ébauche de Galatea, un pont MIDI qui permet d'utiliser tablettes graphiques, souris et autres interfaces de pointage comme interfaces de contrôle MIDI. Le code source est disponible sur ce répertoire.

Si vous êtes arrivée ici par hasard, n'hésitez pas à revenir dans quelques semaines ! L'interface devrait être améliorée et bien documentée d'ici là.

L'élément canvas n'est pas compatible avec votre navigateur.
